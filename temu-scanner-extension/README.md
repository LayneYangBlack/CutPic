# Temu订单扫码助手

Chrome扩展，用于Temu商家后台快速查看订单定制内容。

## 功能特性

✅ **扫码枪输入监听** - 自动识别扫码枪快速输入  
✅ **订单智能匹配** - 支持订单号(WB)、备货母单号(WP)、定制SKU匹配  
✅ **自动打开定制页** - 扫码后直接打开定制内容页面，查看用户上传的图片  
✅ **订单高亮** - 自动高亮匹配的订单行3秒，方便确认  

## 安装步骤

### 1. 准备图标（可选）
如果需要自定义图标，可以：
- 在线转换 `icon.svg` 为 `icon.png` (128x128)
- 或使用任意 128x128 的PNG图片命名为 `icon.png`

### 2. 加载扩展到Chrome

1. 打开Chrome浏览器
2. 地址栏输入：`chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `temu-scanner-extension` 文件夹
6. 完成！

## 使用方法

1. 打开Temu商家后台订单页面：  
   https://agentseller.temu.com/stock/fully-mgt/order-manage-custom

2. 使用扫码枪扫描订单条码（订单号、备货母单号或定制SKU）

3. **自动打开定制内容页面**，直接查看用户上传的图片

4. 订单行会高亮3秒，方便确认

## 技术说明

- **扫码枪识别原理**：监听键盘输入，识别短时间内（<100ms）连续输入+回车的特征
- **订单匹配**：在页面DOM中查找包含扫码内容的订单行
- **数据提取**：使用正则表达式提取订单号、SKU等关键字段
- **UI注入**：动态创建弹窗，不影响原页面功能

## 注意事项

⚠️ 仅在Temu商家后台订单管理页面生效  
⚠️ 需要保持页面已加载订单列表  
⚠️ 扫码枪需配置为键盘输入模式（HID模式）

## 故障排查

**问题：扫码后没有反应**
- 检查是否在正确的页面（订单管理页面）
- 确认扫码枪配置为键盘输入模式
- 打开开发者工具(F12)查看Console是否有错误

**问题：找不到订单**
- 确认扫描的条码内容是订单号(WB)、备货母单号(WP)或定制SKU
- 检查订单是否在当前页面列表中
- 尝试手动输入订单号测试
